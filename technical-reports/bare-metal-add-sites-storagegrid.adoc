= 新增站台至 StorageGRID
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
瞭解如何將站台新增至 StorageGRID 、以提高可靠性和儲存容量。

安裝 StorageGRID 時、您必須至少建立一個站台。您可以建立額外的站台、以提升StorageGRID 您的作業系統的可靠性和儲存容量。

若要新增站台、請完成下列步驟：

.步驟
. 在「站台」頁面上、輸入站台名稱。
. 若要新增其他網站、請按一下最後一個網站項目旁的加號、然後在新的「網站名稱」文字方塊中輸入名稱。根據網格拓撲的需求新增更多站台。您最多可以新增16個站台。
